Analyzing the 2012 Blizzard Battle.net Security Breach: Causes and Solutions (report)

This document presents a concise analysis of the 2012 security breach at Blizzard Entertainment's Battle.net gaming portal, outlining the compromised user data and the company's response. Drawing on principles from the OWASP Top-10 privacy risks project, potential factors contributing to the breach are explored, including weak passwords, inadequate security measures, and phishing schemes. The document concludes with proactive recommendations to prevent future breaches, emphasizing GDPR compliance, regular software updates, robust authentication, employee training, data encryption, security assessments, and need-based access controls. Overall, it provides a comprehensive overview of the incident, its potential causes, and actionable steps for enhancing data privacy and security.

European Electronic Toll Service case study (report)

This report thoroughly examines the European Electronic Toll Service, focusing on user-server interaction, potential risks, and privacy principles. In the user domain, vehicle identification occurs through plates or QR codes, with GPS determining road fees. Servers store transaction logs and billing data. Potential issues include client tool errors, data storage challenges, data sharing concerns, server problems, data retention, and processing errors. Risk analysis reveals various scenarios, underscoring the need for comprehensive solutions. Finally, the report advocates six privacy principles, such as minimizing data collection, disclosure, and centralization, with practical measures like using multiple databases and actively involving users in data processing to enhance system security.
